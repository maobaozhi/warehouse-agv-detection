# Warehouse AGV Detection

Code repository for the paper submitted to The Visual Computer.

## Status

**Code will be released upon paper acceptance.**

This repository serves as a permanent link for the manuscript under review. The complete implementation, including source code, trained weights, and documentation, will be made publicly available following formal acceptance of the paper.

## Paper Information

- **Journal**: The Visual Computer
- **Topic**: AGV detection in warehouse scenarios
- **Dataset**: Warehouse-ijyxt (publicly available at https://universe.roboflow.com/warehouse-ijyxt/warehouse-ijyxt)

## Planned Contents

Upon acceptance, this repository will include:

- Training and inference code
- Pre-trained model weights
- Environment configuration (requirements.txt)
- Documentation and usage instructions
- Reproducible experimental protocols

## Dataset

The Warehouse-ijyxt dataset is publicly available under CC BY 4.0 licence and can be accessed through Roboflow. Dataset statistics and partitioning details are provided in the paper.
